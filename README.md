## 📖 简介

本项目是针对 **山东大学工程软件专业（全国首设）** 的自主维护学习仓库。

作为一个具有鲜明“工程+软件”融合特色的专业，课程体系涵盖了计算思维（Python积分/数据统计）、FastCAE工具链配置等独特内容。鉴于在通用技术社区难以找到针对本专业的特定解决方案，本仓库旨在建立一个**系统性、长期性**的学习支持平台。

> **核心目标**：全面理解专业特点、降低学习门槛、减少重复踩坑。

## 🌟 特性

本仓库主要收录并整理以下内容：

- 🧩 **课堂事例代码**：核心课程的代码实现与重构。
- 🛠️ **专业工具配置**：如 FastCAE 等专业软件的安装与环境配置指南。
- 📚 **特色课程知识**：工程软件特有的理论与实践笔记。
- 💡 **问题解决方案**：作业思路解析、常见错误（踩坑）总结与调试技巧。

## 📂 目录结构

```text
SDU-ES-self-learning/

│   ├── 计算思维/    # 计算思维

```


## 🚀 快速开始

### 📥 获取项目

```bash
git clone https://github.com/zhaobowen/sdu-es-self-learning.git
cd sdu-es-self-learning
```

### 💡 使用建议

1. 进入 `courses` 目录查找对应课程的示例代码。
2. 遇到环境配置问题，请优先查看 `tools` 目录下的配置指南。
