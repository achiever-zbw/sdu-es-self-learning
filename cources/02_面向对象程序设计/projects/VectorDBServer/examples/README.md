# VectorDBServer 测试示例

本目录包含了VectorDBServer向量数据库的测试示例，帮助您快速了解和使用向量数据库的基本功能。

## 快速开始

### 1. 启动服务器

首先确保已经编译并启动了VectorDBServer：

```bash
# 在项目根目录下
cd build
./vectorDB_server
```

服务器将在8080端口启动。

### 2. 使用curl脚本测试

我创建了两个简单的测试脚本：

#### 插入向量示例

```bash
# 进入curl目录
cd examples/curl

# 运行插入脚本
./insert.sh
```

该脚本将插入多个向量到数据库中，包括：
- 一维向量：[0.88], [0.99], [1.01]
- 高维向量：128维随机向量
- 相似向量组：基于[1.0, 2.0, 3.0]的5个变体

#### 搜索向量示例

```bash
# 运行搜索脚本
./search.sh
```

该脚本将执行多种搜索操作，包括：
- 搜索向量[0.91]的最近邻（与README.md示例一致）
- 搜索不同维度的向量

### 3. 预期结果

搜索向量[0.91]的预期响应：
```json
{"vectors":[1,2],"distances":[0.0009000018471851945,0.0063999975100159645],"retCode":0}
```

这表示向量[0.91]的最近邻是ID为1和2的向量，距离分别为0.0009和0.0064。
